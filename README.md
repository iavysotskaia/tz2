## Основные моменты 
Код состоит из нескольких функций, которые находят максимум, минимум, сумму и произведение чисел в массиве. 

Тесты написаны в классе BasicTest

## Ввод и вывод
Ввод находится в файле file.txt \
Вывод производится в консоль
